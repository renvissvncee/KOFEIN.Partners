# Дизайн-система KOFEIN.Partners v2.0

Дизайн-система для React + TailwindCSS v4 с поддержкой тёмной темы.

## 🎨 Цветовая палитра

### Светлая тема
| Название | Значение | Описание |
|----------|----------|----------|
| `--primary` | `#22d3ee` | Основной цвет (cyan) |
| `--accent` | `#fbbf24` | Акцентный цвет (yellow) |
| `--background` | `#f8fafc` | Фон страницы |
| --surface` | `#ffffff` | Фон карточек |

### Тёмная тема (по умолчанию)
| Название | Значение | Описание |
|----------|----------|----------|
| `--background` | `#0a0a0f` | Глубокий тёмный фон |
| `--surface` | `#1a1a24` | Тёмный фон карточек |
| `--card` | `#121218` | Фон панелей |
| `--primary` | `#22d3ee` | Яркий акцент (cyan) |
| `--accent` | `#fbbf24` | Тёплый акцент (yellow) |

## 🌗 Переключение темы

В `App.jsx` реализован переключатель темы:

```jsx
const [darkMode, setDarkMode] = useState(true);

useEffect(() => {
  if (darkMode) {
    document.documentElement.classList.add('dark');
  } else {
    document.documentElement.classList.remove('dark');
  }
}, [darkMode]);
```

Кнопка с иконками `Sun` / `Moon` переключает тему.

## 🚀 Готовые компоненты

### Кнопки
```jsx
<button className="btn-primary">Основная кнопка</button>
<button className="btn-accent">Акцентная кнопка</button>
```

**Особенности:**
- Градиентный фон
- Анимация при наведении (подъём)
- Тень и эффекты

### Карточки
```jsx
<div className="card">Контент карточки</div>
```

**Особенности:**
- Hover-эффект (подъём + свечение)
- Граница меняет цвет при наведении
- Плавные переходы

### Инпуты
```jsx
<input className="input" placeholder="Введите текст" />
```

**Особенности:**
- Фокус с подсветкой primary
- Плавные переходы

## 🎯 Адаптивность

Все компоненты полностью адаптивны:
- `sm:` — мобильные (640px+)
- `md:` — планшеты (768px+)
- `lg:` — десктопы (1024px+)

Пример адаптивной сетки:
```jsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
```

## ✨ Анимации

```css
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

.animate-fade-in {
  animation: fadeIn 0.3s ease-out forwards;
}
```

Используется для плавного появления элементов.

## 📊 Структура страниц

- **Header** — липкий хедер с градиентом
- **Main** — основной контент с отступами
- **Footer** — подвал с копирайтом

## 🎨 Иконки

Используются иконки из `lucide-react`:
- `Coffee` — заказы
- `Package` — меню
- `Settings` — настройки
- `Clock` — время
- `User` — сотрудник
- `Crown` — владелец
- `Moon` / `Sun` — тема

## 💡 Советы по использованию

1. **Не используйте инлайн-стили** — все стили через классы
2. **Используйте CSS-переменные** — для кастомизации
3. **Следуйте сетке** — `max-w-7xl mx-auto` для контейнеров
4. **Добавляйте анимации** — `animate-fade-in` для новых элементов
5. **Проверяйте контраст** — текст всегда читаем
